# E-Module OOP — Pemrograman Berorientasi Objek

E-module interaktif untuk mata kuliah **Pengembangan Bahan Pembelajaran Berbasis Online (PBPBO)** — Modul 2: Pemrograman Berorientasi Objek (Python).

## Fitur

- **3 halaman materi** dengan progress bar navigasi
- **Syntax highlighting** pada semua blok kode Python
- **Tombol salin kode** — satu klik langsung tersalin
- **Glosarium hover bilingual** — istilah Inggris + penjelasan Bahasa Indonesia
- **Infografis pemetaan istilah** OOP (OOP, MRO, DRY, SRP, UML, API)
- **Kotak Tahukah Kamu?** dengan contoh nyata (Tokopedia, Mobile Legends, GoPay)
- **Ringkasan 4 pilar OOP** — Encapsulation, Inheritance, Polymorphism, Abstraction
- **Navigasi sidebar** dengan indikator progres per halaman
- **Responsif** untuk mobile dan desktop

## Materi

## Cara Menjalankan

### Lokal
Buka file `index.html` langsung di browser — tidak perlu server.

### GitHub Pages
1. Fork / upload repo ini ke GitHub
2. Buka **Settings** → **Pages**
3. Source: pilih branch `main`, folder `/ (root)`
4. Klik **Save** — tunggu 1-2 menit
5. Akses di: `https://username.github.io/nama-repo/`

## Teknologi

- HTML5 / CSS3 / Vanilla JavaScript
- Font: [Sora](https://fonts.google.com/specimen/Sora) + [Space Mono](https://fonts.google.com/specimen/Space+Mono) via Google Fonts
